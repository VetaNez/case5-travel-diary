## Описание
Web-приложение "Дневник путешествий" с функциями записи и просмотра путешествий.

## Функционал
- Регистрация и вход пользователей
- Создание записи о путешествии
- Просмотр путешествий других пользователей
- Местоположение с привязкой к геопозиции
- Загрузка изображений мест
- Учёт стоимости путешествия
